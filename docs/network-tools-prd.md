## Executive Summary

Networking utilities and tools for diagnosing network connectivity issues. Initial focus on investigating why a Windows 11 PC on the network is not reachable despite showing an active IP address.

### Goals

2. Identify why specific hosts are unreachable
3. Provide actionable diagnostic output
